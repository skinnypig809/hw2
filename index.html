Redesign Jahunger

Part 1:

This page demonstrate the process of improving this webpage: jahunger's ordering page. The reason I chose this website is that I used this website for few times as a user and felt like it could be improved.

Screenshots of this website:

Here are some problems I found with this website:

1. The submenu on the top "store, location, contact us" is not obvious to see and people usually ignore them if not looking carefully.

2. The submenu for choosing appetizers and entree should always appear in the page so we don't need to scroll all the way up to switch.

3. The search button is not obvious and lack of description.

4. The functionality of "Buy Now" button is adding 1 item to shopping cart, what about adding three cups of cokes? It's annoying to go back and click again

5. The banner image is really large and unnessesery. Especially for mobile users, they need to scroll all the way to the bottom every time when they open up this page

6. The featured products section only has gift cards which can be dealt with better

7. The menu seems a little bit too spread out

Here are some problems WebAIM WAVE found:

1. Some empty links

2. Contrast issues

3. When selecting item, it has redundant link issues because the texts and images are going to the same pages

I think the first two suggestions are good but the third one is not. Because diffrent users has different habits where some of them tend to click text and others tend to click the images. Setting links for both of them is a reasonablt approch.
